
| Begriff                             | Bounded Context    | Definition                                                                                                                                                                | Aliase / Verwandte Begriffe            |
| ----------------------------------- | ------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------- |
| **Benutzerkonto**, **User Account** | Benutzerverwaltung | Der zentrale, administrative Datensatz für eine Person im System. Es enthält eine eindeutige ID, Anmeldeinformationen und ist mit einer oder mehreren `Rollen` verknüpft. | Account, User                          |
| **Rolle**, **Role**                 | Benutzerverwaltung | Eine zugewiesene Berechtigungsstufe, die bestimmt, welche Aktionen ein `Benutzerkonto` im System ausführen kann (z.B. die Rolle `Lehrer` oder `Schüler`).                 | Berechtigungsgruppe                    |
| **Klasse**, **Course**              | Benutzerverwaltung | Eine organisatorische Gruppe, die `Benutzerkonten` mit der `Rolle` `Schüler` zusammenfasst. Dient der Zuordnung von `Kursen`.                                             | Lerngruppe                             |
| **Schüler**, **Student**            | Benutzerverwaltung | Ein `Benutzerkonto` mit der `Rolle` `Schüler`, das einer `Klasse` zugeordnet ist. Repräsentiert die administrative Identität.                                             |                                        |
| **Lehrer**, **Teacher**             | Benutzerverwaltung | Ein `Benutzerkonto` mit der `Rolle` `Lehrer`, das erweiterte Rechte zur Erstellung und Verwaltung von Inhalten hat.                                                       |                                        |
| **Lehrer**, **Teacher**             | Unterrichten       | Der **Autor**, der `Lerneinheiten` und `Kurse` erstellt und deren Struktur festlegt.                                                                                      | Autor, Ersteller                       |
| **Lehrer**, **Teacher**             | Diagnostik         | Der **Analyst**, der aggregierte Leistungsdaten interpretiert, um den Lernfortschritt zu verstehen.                                                                       | Beobachter                             |
| **Schüler**, **Student**            | Lernen             | Der **Lernende**, der eine `Aufgabe` bearbeitet und `Abgaben` erzeugt. In diesem Kontext primär durch seine ID repräsentiert.                                             | Lernender                              |
| **Kurs**, **Course**                | Unterrichten       | Eine organisatorische Hülle, die `Lerneinheiten` in einer bestimmten Reihenfolge für eine `Klasse` zusammenfasst und die `Freigaben` steuert.                             | Lehrveranstaltung                      |
| **Lerneinheit**, **Unit**           | Unterrichten       | Ein wiederverwendbarer, thematischer Inhaltsblock, der `Abschnitte` in einer definierten Reihenfolge enthält.                                                             | Modul, Themenblock, Unterrichtseinheit |
| **Abschnitt**, **Section**          | Unterrichten       | Eine thematische Unterteilung einer `Lerneinheit`, die `Material` und `Aufgaben` gruppiert. Ist die kleinste Einheit für eine `Freigabe`.                                 | Kapitel, Lektion                       |
| **Material**, **Material**          | Unterrichten       | Passiver, nicht-interaktiver Lerninhalt (z.B. Text, Video, Dokument) innerhalb eines `Abschnitts`.                                                                        | Inhalt, Ressource                      |
| **Aufgabe**, **Task**               | Unterrichten       | Eine interaktive Tätigkeit innerhalb eines `Abschnitts`, die eine `Abgabe` erfordert und `Kriterien` zur `Analyse` enthält.                                               | Übung                                  |
| **Kriterium**, **Criteron**         | Unterrichten       | Ein spezifischer Aspekt einer `Aufgabe`, anhand dessen eine `Abgabe` durch die KI analysiert wird (z.B. "Grammatikalische Korrektheit").                                  | Analysekriterium                       |
| **Freigabe**, **Release**           | Unterrichten       | Die Handlung des `Lehrers`, einen `Abschnitt` innerhalb eines `Kurses` für die Schüler sichtbar zu machen.                                                                | Sichtbarkeit schalten                  |
| **Abgabe**, **Submission**          | Lernen             | Eine unveränderliche Aufzeichnung der Lösung eines `Schülers` für eine `Aufgabe`. Löst eine `Analyse` aus und ist mit `Feedback` verbunden.                               | Einreichung, Versuch                   |
| **Analyse**, **Analysis**           | Lernen             | Das strukturierte Ergebnis, das die KI für eine `Abgabe` basierend auf den `Kriterien` generiert. Die Grundlage für das `Feedback`.                                       | Auswertung, Bewertung                  |
| **Rückmeldung**, **Feedback**       | Lernen             | Die KI-generierte, formative Rückmeldung, die ein `Schüler` zu einer spezifischen `Abgabe` erhält.                                                                        | Rückmeldung                            |
